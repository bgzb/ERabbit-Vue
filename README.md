BunnyFresh - Vue 3 E-commerce Full-stack Project

BunnyFresh is a high-performance, responsive e-commerce platform built with the latest Vue 3 ecosystem. It covers the complete shopping lifecycle from product discovery to secure checkout, featuring complex UI components and state management.

✨ Key Features
🛍️ Complete Shopping Flow: Category browsing, product details, shopping cart, and order settlement.

🔍 Advanced UI Components: * Image Zoom: High-precision product magnification using Template Refs.

SKU Algorithm: Dynamic stock and price calculation based on attribute selection.

Infinite Scroll: Smooth data loading for long product lists.

🔐 Auth & Security: Persistent login state with JWT and automated Token refreshing via Axios interceptors.

⚡ Performance Optimized: Image lazy-loading and component-based skeleton screens for faster First Contentful Paint (FCP).

💳 Payment Integration: Simulated checkout process supporting multi-platform payment logic.

🛠️ Tech Stack
Category	Technology
Framework	Vue 3 (Composition API)
Build Tool	Vite
State Management	Pinia (with Persistence Plugin)
Routing	Vue Router 4
Language	TypeScript
UI Library	Element Plus
HTTP Client	Axios
Utilities	VueUse, Day.js

Bash
